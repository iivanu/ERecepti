# eRecipe
Website for Human Computer Interaction course

Link on Netify: https://erecipe.netlify.app

# Usage
a) Debuging
1. git clone git@github.com:iivanu/eRecipe.git
2. npm install
3. gatsby develop
4. open localhost:8000

b) Build site
1. git clone git@github.com:iivanu/eRecipe.git
2. npm install
3. gatsby build
4. Data is stored in public folder of the project
